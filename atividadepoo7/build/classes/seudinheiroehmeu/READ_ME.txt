sistema simples de contas bancarias, com as funções dde fazer saques, fazer depositos e gerar extratos de movimentações bancárias. Contendo as opções de conta:

1- Conta poupança simples;
2- Conta corrente sem cartão de crédito;
3- Conta corrente com cartão de crédito;

neste sistema, ao abrir a conta, os dados do cliente que devem ser informados são:
1- nome;
2- endereço;
3- telefone;
4- E-mail; 
5- CPF;
6- idade;
7- tipo de conta;
assim como os dados específicos de cada tipo de conta. Que são:
Conta corrente:
	1- porcentagem de rendimento da conta;
	2- data de vencimento do deposito (o dia em que serão calculados os acrescimos na conta);

conta Corrrente:
	1- renda bruta(pois o sistema só irá aceitar criar contas para valores que sejam superiores ao minimo informado na criação do sistema);
	

conta Corrente com cartão de crédito:	
	1- renda bruta(pois o sistema só irá aceitar criar contas para valores que sejam superiores ao minimo informado na criação do sistema);
	2- limite do cartão de crédito;
